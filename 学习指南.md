# Rust 学习指南

这是一个面向 Rust 初学者的自学仓库。

目标不是“把语法点堆满”，而是按阶段建立稳定的 Rust 心智模型，尤其是：

- Rust 为什么这样设计
- 所有权、借用、生命周期到底解决什么问题
- 如何把你已有的 JavaScript、Python、Java 经验迁移到 Rust
- 如何从“能看懂”走到“能自己写”

当前进展已经从“入门阶段铺设”进入到“初级阶段起步”：

- 初始化仓库级 `AGENTS.md`
